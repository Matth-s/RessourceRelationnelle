# RessourceRelationnelle

Pour la partie backend lors du clone du projet
  - se deplacer dans backend -> ResourceAPI :  dotnet ef database update

Pour le frontend admin :
 Aller dans le dossier admin-frontend 
  - npm i (installation des dépendances de node)
  - créer un fichier .env a la racine du admin-frontend
      Ajouter cette ligne: VITE_PUBLIC_API_ROUTE="https://localhost:7090/api/" // Remplacer le port si nécessaire

Pour la version de node : v24.11.1
Version de npm : 11.6.2
